What’s Summoners War Statistics Web
Summoners War Statistics Web is a website which stores, analyse and shows in a friendly way contributors data but also implements the most important features from Summoners War Statistics Windows App which had been discontinued because of technology limits.
Data Logs Section
In Data Logs Section you can find analysis, charts, tables and reports which uses whole data stored in database, i.e.:
  • Wizard (Summoner)
  • Guild
  • Monsters
  • Runes
  • Artifacts
  • Monster builds
  • RTA Monster builds
  • Dungeon runs
  • Rift runs
  • Raid runs
Data send through Data Logs Section IS stored in our database.
Personal Section
In Personal Section you can find towers & flags calculator, dimension hole calculator (2A monsters) and graphic representation of your profile:
  • Wizard (Summoner)
  • Monsters (with details)
  • Runes (with details)
  • Guild
  • Guild members activity
  • Friends activity
Data send through Personal Section IS NOT stored in our database.
Data Logs Section
Every data used in Data Logs Section is being collected annonymously by using website built-in Upload Functionality (together with Rank & Compare) or by using SWEX Plugin. More information in Contribute page.
Personal Section
Every data used in Personal Section is not anonymous but it’s not stored in Summoners War Statistics Database. It works similar to all known SWOP - locally. Data is stored in Browser Local Storage to not force you to upload your profile in Personal Section everytime you visit it.
I don't want my data to be there
I can assure you that no user will be able to tell if specific monster, rune, artifact or dungeon run is yours or not. However, if you want your data to be deleted, please contact me through Reddit (as for now the only form of contact) and we will see what can be done. Be prepare to give me some additional info I ask (most of the time example JSON file from your profile is enough), no personal data involved.
